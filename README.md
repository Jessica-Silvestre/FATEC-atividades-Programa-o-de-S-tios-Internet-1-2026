# Beleza na Fatec
Aplicação web de marketplace de produtos de beleza com **HTML, CSS e JavaScript puro**.

## Funcionalidades
- Catálogo de produtos com filtros
- Cadastro e login de usuários
- Gerenciamento crud (criar, editar, deletar, publicar)
- Design responsivo

## Tecnologias
- HTML5 | CSS3 | JavaScript Vanilla
- API REST (DummyJSON + Render)

## Como Usar

1. Abra `index.html` no navegador
2. Navegue pelo catálogo de produtos
3. Clique em "Cadastro de Produtos" para:
   - Cadastrar como novo usuário
   - Fazer login
   - Adicionar e publicar produtos

## Fluxo
Catálogo → Cadastro → Login → Meus Produtos → Publicar → Catálogo
